Яндех Мобилизация - расписание лекций в школах Яндекса.

В этой странице можно найти информацию о предстояших, а также прошедших лекциях. Кроме этого можно посмотреть материалы прошедших лекций и узнат более подробную инфромацию о лекторах школ.

Некоторые обяснения коду даны в комментариях в коде.

Использовы:
Sass препроцессор: sass-lang.com
SMACSS - архитектура или style guide для конструкции файлов на сайте: smacss.com
jQuery - jquery.com
jQuery MatchHeight - плагин для уравнения высоты элементов: https://github.com/liabru/jquery-match-height
jQuery Modal - минимальный плагин для сождания всплывающих окон: https://github.com/kylefox/jquery-modal
jQuery Selectize - стили для select форм: https://github.com/selectize/selectize.js
Font Awesome - иконки: fontawesome.io
